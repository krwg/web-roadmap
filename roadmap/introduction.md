# Введение

Добро пожаловать в **web-roadmap** — 22-недельный путь к junior full-stack разработчику.

## Главная цель

Перестать быть «вайбкодером» и стать инженером, который **понимает, почему код работает**, и может переписать его без ИИ.

**Главный враг — иллюзия компетентности.** Туториал посмотрел — кажется, всё ясно. Мы ломаем эту иллюзию **ежедневной практикой**.

## Принцип прогрессии

```
Наглядное → Логика → Абстракции → Системы
HTML/CSS  →  JS    →  React/TS  →  Full-Stack + DevOps
```

Не перепрыгивайте: слабый фундамент в HTML/JS превратит React и бэкенд в мучение.
